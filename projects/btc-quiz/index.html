The Bitcoin Quiz App

A web app offering a selection of Bitcoin related quizzes. The application was built using the Angular framework leveraging the Bootstrap library for the user interface.

Launch ApplicationView Repo

Objective

This project served as my introduction to the Angular framework and my primary objective was to provide a simple and clean user interface for a simple web app.

The application would offer a selection of multiple choice quizzes. Once a quiz is complete the user will be shown the results and offered to pick another quiz.

The Solution

A clean and simple UI offering Bitcoin quizzes at three difficulty levels.
