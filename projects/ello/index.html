ello*

ello* is a progressive web application integrating with the Meetup API to provides users with access to all the Meetup events. Browse through events in your area or search events in any city across the globe.

Launch ApplicationView Repo

Objective

Ello is designed to offer Meetup event search related features in the form of a progressive web application (a site that offering advanced features such as offering a native application feel)

The Solution

A test driven development approach was followed to build this prototype which interacts with the Meetup API using a serverless framework.

The web app provides a filterable list of events that can be selected for further details.
